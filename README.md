# Cloudreve Flutter移动客户端
一款使用Flutter开发的移动客户端，新手勿喷

 * 支持多文件 上传 下载
 * 暗黑模式(跟随系统或者自行设置)
 * 在线预览图片
 * 实时分享文件, 查看已分享以及设置分享
 * 查看已设置的WebDav
 * 搜索文件, 并打开文件所在路径
 * 暂时由于设备原因只支持Android, iOS用户暂时无法使用
 * 暂时只支持单一主机，从机账户无法上传
